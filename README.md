# UPT.Script (Quiz Solver)

## Hotkeys

| Combinație | Acțiune |
|------------|---------|
| `Ctrl+Alt+S` | Request din clipboard |
| `Ctrl+Alt+D` | Captură zonă → **OCR** → model FAST |
| `Ctrl+Alt+C` | Captură zonă → **imagine** → model VLM |
| `Ctrl+Shift+A` | Setare cheie API |

Tragi dreptunghi pe ecran (chenar punctat). **Esc** = anulare. Fără meniu extra.

Setări: `application_settings.json`
