# UPT.Script (Quiz Solver)

## Hotkeys

| Combinație | Acțiune |
|------------|---------|
| `Ctrl+Shift+S` | Request din clipboard (text/imagine copiată) |
| `Ctrl+Alt+D` | Captură zonă — chenar punctat, fără fundal negru |
| `Ctrl+Shift+A` | Setare cheie API |

Captură: tragi dreptunghi pe ecran. **Esc** sau click dreapta = anulare.

Setări: `application_settings.json`
